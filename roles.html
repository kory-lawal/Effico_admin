Roles (4)

Owner (you)
This role has full access to all the features and permissions of the admin workspace and can change company profile and security.
Admin
This role has full access to all the analytics and can manage users features including creating new users, assigning tokens, inviting new members etc.
Moderator
This role has full access to the content management. Can publish new articles, delete or edit current articles in the dashboard.
Read-only
This role has read-only access to the users statistics and analytics. Can also read users activities and chat history.
